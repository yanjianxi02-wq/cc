# 茵曼达人画像分析网站

这是可上传到 GitHub Pages 的静态网站版本。

## 文件结构

- `index.html`：GitHub Pages 首页跳转入口。
- `web/index.html`：网站主页面。
- `web/app.js`：页面交互与数据读取。
- `web/styles.css`：页面样式。
- `web/assets/inman-nature-bg.png`：背景图。
- `outputs/qly_creators/latest_creator_pool.json`：达人画像数据。
- `outputs/qly_creators/latest_creator_hot_products.json`：达人近14天女装 TOP50。
- `outputs/bestseller_db/latest_bestseller_db.json`：BI 爆款数据库。

## GitHub Pages 使用

将本目录所有文件上传到 GitHub 仓库根目录，然后在 GitHub Pages 中选择从 `main` 分支根目录发布。

注意：本目录包含当前业务数据快照。公开发布前请确认这些销售数据、达人数据和款式数据可以对外公开。
